# Sistema de Gestão de Participantes para Eventos

##  Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de auxiliar na gestão de participantes de eventos.

O sistema permite cadastrar, listar e remover participantes através de um menu interativo no terminal, armazenando os dados em um arquivo JSON para que as informações não sejam perdidas ao fechar o programa.

Este é um dos meus primeiros projetos práticos desenvolvidos sem seguir um tutorial completo, aplicando conceitos de lógica de programação, funções, manipulação de arquivos e organização de código.

---

##  Funcionalidades

* Cadastro de participantes
* Listagem de participantes cadastrados
* Remoção de participantes por CPF
* Persistência de dados utilizando JSON
* Menu interativo no terminal
* Organização do código em múltiplos arquivos
